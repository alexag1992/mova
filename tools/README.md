# Генерация аудио для МОВА

## Установка
```bash
pip install -r requirements.txt
```

## Запуск
```bash
cd tools
python generate_audio.py
```

## Что делает скрипт
- Читает все JSON файлы из папки `data/`
- Генерирует MP3 для каждого белорусского слова и фразы
- Сохраняет в папку `audio/` с нужной структурой
- Пропускает уже существующие файлы (можно запускать повторно)

## Структура
```
audio/
├── alphabet/          — озвучка букв
├── lesson-01/         — аудио урока 1
├── lesson-02/         — аудио урока 2
├── ...
└── lesson-24/         — аудио урока 24
```

## Примечания
- Используется украинский язык (`uk`) как ближайший к белорусскому
- Google может временно заблокировать при слишком частых запросах
- Между запросами пауза 0.5 сек
- Генерация всех файлов занимает ~30-40 минут
